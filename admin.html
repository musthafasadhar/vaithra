Google Sheets Email Setup

One-time setup to save subscriber emails automatically to a Google Sheet with timestamp.

Step 1 — Create a Google Form

Go to forms.google.com → Blank form → Add one question: Email Address (Short answer, required).

Step 2 — Get Form Action URL

In the form → 3-dot menu → Get pre-filled link → type any email → Get link. In the URL bar, change /viewform to /formResponse. That's your FORM_ACTION.

Step 3 — Get the Entry ID

In that same pre-filled URL, find entry.XXXXXXXXXX — that number is your ENTRY_ID. Open daily-drop.html, find the two lines near the bottom and paste both values in.

Step 4 — Connect to Google Sheets

In your Google Form → Responses tab → click the green Sheets icon → Create spreadsheet. Every subscriber email now auto-saves with a timestamp. Done ✓

Step 5 — Set Firestore Security Rules

In Firebase Console → Firestore → Rules tab → paste this and click Publish:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /drops/{dropId} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
